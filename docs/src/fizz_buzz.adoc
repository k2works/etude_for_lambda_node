= FizzBuzz

== 仕様
* ３で割り切れる場合は「Fizz」を出力する。
* ５で割り切れる場合は「Buzz」を出力する。
* 両者で割り切れる場合は「FizzBuzz」を出力する。

== 設計
=== TODOリスト
* [*] 1ならば１を返す
* [*] 2ならば２を返す
* [*] 3ならばFizzを返す
* [*] 6ならばFizzを返す
* [*] 5ならばBuzzを返す
* [*] 10ならばBuzzを返す
* [*] 15ならばFizzBuzzを返す
* [*] 30ならばFizzBuzzを返す
